Python Text-Based Slot Machine

Built this Python mini-game to practice functions, variables, loops and conditional statements!

Instructions:

1) Add initial deposit.
2) Select Enter to play.
3) Choose the number of lines to bet on.
4) Input the value you wish to bet on each line, the bet is multiplied according to how many lines you have bet on.
5) Outcome of the spin is displayed on the console.
6) Press Enter to spin again or "q" to exit the game.
